# Contributing

English: [CONTRIBUTING.md](CONTRIBUTING.md)

## Vor einem Pull Request

- Änderungen sollten sich auf die Flutter-App, die Fragendaten oder den Release-Workflow konzentrieren.
- Aktualisiere `README.md`, `README.DE.md` oder das Changelog, wenn sich Verhalten ändert.
- Ergaenze oder aktualisiere Tests, wenn Quiz-Logik, Persistenz oder Theming angepasst werden.

## Lokale Validierung

```bash
flutter analyze
flutter test
```
