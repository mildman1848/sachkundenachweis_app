# Security Policy

English: [SECURITY.md](SECURITY.md)

## Meldung einer Schwachstelle

Bitte Sicherheitsprobleme zuerst privat melden, bevor ein öffentliches Issue erstellt wird.

## Umfang

- lokale Datenspeicherung
- eingebundene Fragendaten
- Release-Automatisierung
- Drittanbieter-Abhaengigkeiten der Flutter-App
