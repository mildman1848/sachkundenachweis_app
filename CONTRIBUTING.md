# Contributing

Deutsch: [CONTRIBUTING.DE.md](CONTRIBUTING.DE.md)

## Before Opening a Pull Request

- Keep changes focused on the Flutter app, question data, or release workflow.
- Update `README.md`, `README.DE.md`, or the changelog when behavior changes.
- Add or update tests when quiz logic, persistence, or theming changes.

## Local Validation

```bash
flutter analyze
flutter test
```
