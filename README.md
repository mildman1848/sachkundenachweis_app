# Sachkundenachweis NRW Learning App

Deutsch: [README.DE.md](README.DE.md)

Flutter learning app for the dog ownership certification exam in North Rhine-Westphalia.

## Highlights

- Includes the official question catalog used for the Sachkundenachweis.
- Focuses on adaptive learning, progress tracking, and local-only data storage.
- Supports multiple themes and responsive layouts for phone, tablet, and desktop.
- Ships as a Flutter application with automated release tooling.

## Tech Stack

- Flutter and Dart
- Shared preferences for local progress data
- Charting for progress visualization
- GitHub-based release workflow

## Local Development

```bash
flutter pub get
flutter analyze
flutter test
flutter run
```

## Source Material

The content is based on the official catalog of the Tierarztekammer Nordrhein.
